# Speedify

A Sons of the Forest mod that allows you to adjust game speed.

## Description

Speedify is a simple utility mod that lets you speed up or slow down game time.

## Requirements

### [OptionalDependAPI](https://sotf-mods.com/mods/thirtytwelve/optionaldependapi) 

## Usage
- All in Config menu.
- Toggle with keybind/config/mod
- Adjust the speed multiplier
- Set to automatically disable the speed after duration
